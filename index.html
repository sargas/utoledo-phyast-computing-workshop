UToledo Computing Workshop Series

Physics (like most sciences) makes heavy use of computers without any formal education in computer science or software engineering; we would like to bridge that gap with these workshops. All workshops are at 1:30PM in McMaster 4009.

In-depth introduction to the Linux/UNIX command line

  • Basic introduction to Linux/UNIX commands - June 1st, 2015

    Should be friendly to REU's who haven't used the command line before.

    Extra: Cheat sheet and a briefly used presentation.

    No Video

  • Making the command line shell work for you - June 3rd, 2015

    Will cover tcsh, pipes, aliases, history commands, environmental variables, startup files, wildcards, and anything not covered in the first day.

    Extra: Cheat sheet and presentation.

  • The in-depth portion of this week - June 5th, 2015

    View system calls (with strace/dtruss), compare assembly/compiled/interpreted code, overview of common file system structure, having multiple users in Linux/UNIX, and file permissions.

    Extra: Presentation.

Week of Version Control

  • Version Control Systems - June 16th, 2015

    This will be focused on Git, largely taking from Software Carpentry.

    See the software installation notes, and the workshop website.

  • Git it out - June 18th, 2015

    This workshop will explore using Git collaboratively, to aid in sharing code and contributing changes.

    This is a continuation of the previous day's notes.

Week of Python

  • Introduction to Python - July 7th, 2015

Social Media
